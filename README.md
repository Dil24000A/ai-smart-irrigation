# 🌱 AI Smart Irrigation System

An IoT + AI based smart irrigation system that automates watering using real-time soil data and machine learning.

## 🚀 Features
- Real-time soil moisture monitoring
- AI-based irrigation prediction
- Weather-based automation
- Remote pump control
- Web dashboard

## 🧰 Tech Stack
- ESP8266 (NodeMCU)
- Python (AI Model)
- Firebase Realtime DB
- HTML/CSS/JS Dashboard

## 📊 How it works
1. Sensor reads soil moisture
2. Data sent to Firebase
3. AI predicts watering need
4. Pump auto ON/OFF
